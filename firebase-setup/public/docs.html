üß†codecontextpro remember

Store information in persistent AI memory for future recall

codecontextpro remember "User prefers React hooks over class components"

# With context and type
codecontextpro remember "API uses JWT tokens" --context "auth-system" --type "technical"

Options:

  • -c, --context <context> - Context for the memory (default: "cli-command")
  • -t, --type <type> - Type of memory (default: "general")

üîçcodecontextpro recall

Search and retrieve memories from your persistent storage

# Limit results

Options:

  • -l, --limit <number> - Maximum number of results (default: 10)
